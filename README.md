Projeto de Desenvolvimento Mobile

O projeto foi criado usando o modelo básico do Android Studio, em Java, com configuração padrão.
Na parte visual, organizei a tela com:
    • Uma imagem no topo com a logo 
    • Um campo pra digitar o nome do cliente 
    • Opções de adicionais (Bacon, Queijo e Onion Rings) 
    • Botões de "+" e "-" pra escolher a quantidade 
    • Um espaço mostrando o resumo do pedido 
    • Um botão final "FAZER PEDIDO" 
Também criei um estilo de texto pra manter tudo mais organizado e padronizado na tela.

Na parte do código, implementei a lógica do app para:
    • Controlar a quantidade (sem deixar valores negativos) 
    • Calcular o preço total do pedido 
    • Considerar os adicionais escolhidos 
    • Abrir o e-mail do usuário com o pedido já pronto para envio
